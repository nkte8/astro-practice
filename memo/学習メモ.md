
# レイアウト機能

ページのレイアウトごとtemplate化することで、統一感のあるウェブページを作ることができる機能。

1. レイアウトとして定義したastroファイルを作成する
   レイアウトには `<slot />` タグが埋め込まれている必要がある（slotの箇所に個別のページが差し込まれる）
2. 個別のページについてコンポーネントとして読み込む。以下は`index.astro`の例
```astro
---
import BaseLayout from '../layouts/BaseLayout.astro'; <------- 読み込み
const pageTitle = "Home Page";

const show_dev = true;
---
<BaseLayout pageTitle={pageTitle}> <---------- コンポーネントとして読み込む。変数をコンポーネントに渡す(props)ことも出来る。
{show_dev && <div>develop mode on</div>} <----------- indexのコンテンツ
...
</BaseLayout>
```

レイアウトは入れ子にすることができる。例えばブログページについては
`(個々のmarkdown) → MarkdownLayout → BaseLayout`の入れ子構造となっており、BaseLayoutのレイアウトを保ちながらMarkdownLayoutのレイアウトも保持している。
